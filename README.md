# Ft-JUPYTER Playground

Collection of experiments and tutorials for FastHTML/FastJUPYTER development.

## Experiments

1. **PageManager** - Multi-page FastHTML site manager with dynamic updates
   - Variable tracking
   - ID-based updates
   - HTMX integration
